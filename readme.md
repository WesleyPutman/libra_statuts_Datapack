# 📊 Libra Statuts - Minecraft Datapack

**Libra Statuts** est un datapack complet, léger et vanilla-friendly qui permet aux joueurs de modifier dynamiquement leur rôle et statut de présence (AFK, Occupé, Dispo) en jeu via des commandes d'interface simples (`/trigger`).

---

## ✨ Fonctionnalités

*   **Changement de rôle à la volée** : Indiquez instantanément votre état aux autres joueurs.
*   **Affichage dynamique** : Les rôles apparaissent sous forme de préfixe coloré dans le chat et dans la liste des joueurs (Tabulation).
*   **Aucun droit admin requis** : Utilise le système de `trigger`, permettant à n'importe quel joueur sans permissions de l'utiliser.

---

## 🎭 Statuts Disponibles & Commandes

> 💡 **Astuce :** La commande `/trigger status_menu` ouvre un menu interactif cliquable directement dans le chat pour choisir graphiquement son rôle.

---

## ⚙️ Installation

1. Téléchargez le fichier `libra_statuts_Datapack.zip`.
2. Glissez-le dans le dossier `datapacks` de votre monde (solo ou serveur).
3. Extraire les fichiers.
4. Lancez le monde ou tapez `/reload` dans le chat.
